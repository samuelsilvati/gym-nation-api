 ## RODAR A API

 Instalar as dependencias
 ```sh
 npm install
 ```

Gerar base de dados local
 ```sh
 npx prisma migrate dev
 ```
Gerenciar a base de dados com prisma
```sh
npx prisma studio
```


Rodar a API
 ```sh
 npx run dev
 ```
 
###  API address
```
HTTP server running on http://localhost:3333
 ```
